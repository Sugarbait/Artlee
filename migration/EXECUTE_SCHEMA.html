Automated Migration Assistant

1 Copy SQL to Clipboard
Click the button below to automatically copy the schema SQL to your clipboard.
2 Open Supabase SQL Editor
The Supabase SQL Editor will open in a new tab automatically.
3 Paste and Run
In the SQL Editor, press Ctrl+V (or Cmd+V on Mac) to paste, then click the "Run" button.
4 Run Data Migration
Once the SQL succeeds, click "Complete Migration" below to automatically transfer all data.
